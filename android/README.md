# Android Project - Verum Omnis Legal Forensic AI

This is the native Android project generated by Capacitor for the Verum Omnis application.

## Automatic Building in Android Studio

This project is configured to build automatically when opened in Android Studio. The build process will:

1. **Auto-detect Android SDK**: Android Studio will automatically create `local.properties` with your SDK path
2. **Auto-sync Gradle**: Gradle will automatically download dependencies and sync the project
3. **Auto-configure build tools**: Required Android SDK components will be suggested for installation if missing

## First Time Setup

When you first open this project in Android Studio:

1. **Open the project**:
   - File → Open
   - Navigate to the `android` folder in this repository
   - Click OK

2. **Wait for Gradle sync** (2-10 minutes):
   - Android Studio will automatically start syncing
   - Progress will be shown in the bottom status bar
   - Dependencies will be downloaded automatically

3. **Accept SDK updates** (if prompted):
   - Android Studio may prompt you to install missing SDK components
   - Click "Install" or "Accept" to download them automatically

4. **Build completes automatically**:
   - Once Gradle sync finishes, the project is ready
   - You can run the app on a device or emulator

## Troubleshooting

### Gradle Sync Fails

If Gradle sync fails on first open:

1. **Check Internet connection**: Dependencies need to be downloaded
2. **Click "Sync Project with Gradle Files"**: Top toolbar icon
3. **Try "Invalidate Caches"**: File → Invalidate Caches → Invalidate and Restart

### Missing SDK Components

If Android Studio reports missing components:

1. **Click the provided link** in the error message
2. **Or go to**: Tools → SDK Manager
3. **Install suggested components**

### Build Still Fails

Try a clean build:

1. Build → Clean Project
2. Build → Rebuild Project

Or from command line:
```bash
./gradlew clean
./gradlew build
```
